# Birkirkara FC Website

Modern club website built with Next.js 15, Sanity CMS, Stripe, and Vercel.

## Docs
See `docs/ARCHITECTURE.md` for full stack explanation, agent boundaries,
and environment variable reference.

## Quick start
```bash
cp .env.example .env.local
# fill in your tokens
npm install
npm run dev
```

## Sanity Studio
Visit `http://localhost:3000/studio` to access the content editor.

## Deploy
Push to `main` branch → Vercel auto-deploys.
